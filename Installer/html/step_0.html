This wizard will install Long War: Community Edition, version .

You must install Long War 1.0 prior to running this installer. You should ensure you do not have any other mods installed aside from Long War 1.0, or your game may be corrupted and you will have to download it again.

You will also need to have UPKUtils downloaded and extracted. Make sure to note the location you have extracted to.

Save files created without LWCE will not work with LWCE. You will need to start a new campaign.

Click Next to proceed with installation, or click Cancel at any time to exit without making changes.
